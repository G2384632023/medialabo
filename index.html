「Webコンテンツ作成」のテーマ

世界の天気を表示させるWebコンテンツ作成を行った。 プロジェクト課題で作成したページproject.html

本テーマについて

このテーマでは、HTML・CSS・JavaScriptを用いたWebページ作成を通して、Webの仕組みや動的なコンテンツ作成の基本を学ぶことを目的としています。Web技術の基礎を実践的に学ぶことで、情報発信の力を身につけます。

プロジェクト課題について

プロジェクト課題では、世界の天気を表示するWebアプリケーションを作成しました。OpenWeatherMapのAPIを使って、都市ごとの天気情報を取得し、JavaScriptでページ上に表示する仕組みを実装しました。DOM操作やデータの整形、表示の工夫など、講義で学んだ内容を活かして取り組みました。特に、JavaScriptだけで条件分岐やループを使わずにデータを表示するという制約を設けて、より深くDOMの操作を理解することができました。

プロジェクト課題で作成したページ：project.html

数当てゲーム

課題3-1では、1〜10の中からランダムに選ばれた数を当てるゲームをJavaScriptで作成し、入力に応じて結果を表示する機能を実装しました。ユーザーが入力した数値とランダムに選ばれた数値を比較し、正解かどうかを判定するロジックを組み込みました。さらに、ユーザーが何度も挑戦できるように、リセット機能も追加しました。この課題を通じて、JavaScriptの基本的な文法や条件分岐、ループ処理の使い方を実践的に学ぶことができました。講義資料やオンラインリソースを参考にしながら、試行錯誤を繰り返して完成させたので、非常に達成感がありました。

数当てゲームのページ：kazuate.html

各授業回の学習事項

第1回 WWWとHTML

Webの基本的な仕組みであるWWWについて学び、HTMLを使って見出し、段落、リンク、画像などの基本的なページ要素を記述する方法を習得しました。HTMLの個別の要素やHTML文書全体の構成を学習できたのでいい経験になったと思います。

演習の成果：演習1-4のHTMLファイル

第2回 CSSによる Web ページの装飾

HTMLが最初に与えられ、次にそのページの装飾をするCSSの構成について学習しました。文字の色を指定した色に変更する方法であったり、ページへのリンクの色や下線を追加する方法などを学習しました。

演習の成果：演習2-3のHTMLファイル

第3回 JavaScript の基本

JavaScriptの基本文法や変数、条件分岐、繰り返し処理などを学びました。簡単なスクリプトを使って動的なページを作る練習をしました。

演習の成果：演習3-6のHTMLファイル

第4回 DOM と JavaScript

DOM（Document Object Model）を使って、HTML要素を動的に操作する方法を学びました。JavaScriptを使って、ページの内容を変更したり、新しい要素を追加したりする技術を習得しました。

演習の成果：演習4-10のHTMLファイル

第5回 イベント処理

JavaScriptのイベント処理について学びました。ユーザーの操作に応じて、ページの内容を動的に変更する方法を習得しました。クリックイベントやキーボードイベントなど、様々なイベントを扱う技術を学びました。

演習の成果：演習5-7のHTMLファイル

工夫点，苦心点，強調点，感想

今回の「Webコンテンツ作成」の授業を通して、私はWebの仕組みやHTML・CSS・JavaScriptといった基本的な技術を実践的に学ぶことができました。正直なところ、最初はとても難しそうというイメージが強く、不安もありましたが、実際に手を動かしてコードを書いていくうちに、難しさの中にも少しずつ理解が深まり、楽しさも感じられるようになりました。 特に印象に残っているのは、天気情報を表示するWebアプリケーションを作成したプロジェクト課題です。今もうまく動くか不安で仕方のない、とても難しい課題でした。最初から最後まで本当にできるのかな？と不安でした。しかし、先生やTAの先生に教えていただいて、少しずつ成長を感じられた時には、大きな達成感がありました。 また、数当てゲームの課題では、入力した数字とランダムに選ばれた数字を比較して結果を表示するというシンプルな内容でしたが、条件分岐やイベント処理の理解を深める良い機会になりました。結果の表示方法を工夫したりと、細かい部分にもこだわって作ることで、より実用的なWebページに近づけることができました。 授業の中では、毎回の演習を通じて少しずつステップアップしていく構成になっていたので、初心者の私でも無理なく学習を進めることができました。HTMLでページの構造を作り、CSSで見た目を整え、JavaScriptで動きを加えるという流れを繰り返すことで、Webページがどのように作られているのかを実感できました。 もちろん、うまくいかないこともたくさんありました。思ったように表示されなかったり、エラーが出て原因がわからなかったりして、何度もコードを見直すこともありました。そんな中でも自分で少しずつ解決していく過程がとてもやりがいになりました。自分で考えて試行錯誤することの大切さを改めて感じました。 この授業を通じて、Web技術の面白さと奥深さを知ることができました。今後は、今回学んだ基礎を活かして、もっと複雑なWebアプリケーションにも挑戦してみたいと思います。特に、APIを活用したデータの表示や、ユーザーインターフェースの工夫など、実用的なスキルをさらに磨いていきたいです。将来的には、自分のポートフォリオサイトを作ったり、趣味のWebサービスを公開したりすることも目標にしています。 活用については、さらに深く学んでいきたいです。今回の課題を通じて得た知識や技術を活かして、より高度なWebアプリケーションを作成できるように頑張ります。
