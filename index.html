「Webコンテンツ作成」のテーマ

本テーマについて

このテーマでは、HTML・CSS・JavaScriptを用いたWebページ作成を通して、Webの仕組みや動的なコンテンツ作成の基本を学ぶことを目的としています。

プロジェクト課題について

プロジェクト課題では、自分たちでテーマを設定し、HTMLとCSS、そしてJavaScriptを使ってWebページを作成しました。レイアウトやデザインだけでなく、ユーザーの入力やクリックに反応するインタラクティブな要素も取り入れました。これにより、Web開発の基本的なスキルと実践的な応用力を身につけることができました。

プロジェクト課題で作成したページ：project.html

数当てゲーム

数当てゲームは、ランダムに選ばれた1から100までの数字を、プレイヤーが何度かのヒントを頼りに当てるシンプルなゲームです。ユーザーが入力した数と正解の数を比較し、「大きい」「小さい」といったヒントを表示します。これにより、JavaScriptの条件分岐や変数、イベント処理について実践的に理解を深めることができました。

数当てゲームのページ：kazuate.html

各授業回の学習事項

第1回 WWWとHTML

Webの基本的な仕組みであるWWWについて学び、HTMLを使って見出し、段落、リンク、画像などの基本的なページ要素を記述する方法を習得しました。

演習の成果：演習1-4 - HTMLのさまざまな要素

第2回 CSSによる Web ページの装飾

CSSを用いて、Webページのデザインやレイアウトを調整する方法を学びました。フォント、色、余白、配置などのスタイルを適用し、視覚的に整ったページを作成しました。

第3回 JavaScript の基本

JavaScriptの変数、演算、関数、条件分岐などの基本的な文法とロジックを学び、動的にWebページの内容を変化させる仕組みを理解しました。

第4回 DOM と JavaScript

DOM（ドキュメントオブジェクトモデル）を介してHTMLの要素をJavaScriptで操作する方法を学びました。ページ内のテキスト変更や要素の追加など、動的なページ操作が可能になりました。

第5回 イベント処理

ユーザーのクリックや入力などのイベントに対してJavaScriptで反応させる方法を学び、インタラクティブなWebページを実現しました。
